# Rahul Portfolio Starter (Static)

This is a **single-page, modern portfolio** that you can host for free.

## Quick deploy (GitHub Pages)

1) Create a repo:
- Option A (recommended): `rahulgzb.github.io` (user site)
- Option B: any repo name (project site)

2) Upload these files to the repo root:
- `index.html`
- `Rahul_Mehta.pdf` (optional)
- `assets/`

3) Enable Pages:
- Repo → **Settings** → **Pages**
- Source: **Deploy from a branch**
- Branch: `main` (or `master`) + `/ (root)`

4) Your site will be available at:
- User site: `https://<username>.github.io`
- Project site: `https://<username>.github.io/<repo>`

## Quick deploy (Cloudflare Pages)

- Create a Pages project and connect the GitHub repo, or drag & drop the folder.

## Customize

Open `index.html` and edit:
- Headline + summary
- Projects
- Social links
- Add images under `assets/` and reference them in HTML
